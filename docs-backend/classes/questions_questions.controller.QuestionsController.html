    Class QuestionsController

    Questions controller that handles quiz question retrieval

    This controller provides endpoints for fetching questions by topic with pagination support. All endpoints require JWT authentication and return questions filtered by the user's level.

    // Get questions for a specific topic
    GET /questions?topicId=articles&cursor=some_cursor
    Authorization: Bearer <jwt_token>

    // Response

      "questions": [...],
      "nextCursor": "next_page_cursor",

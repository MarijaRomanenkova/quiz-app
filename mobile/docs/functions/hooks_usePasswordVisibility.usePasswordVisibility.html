Function usePasswordVisibility

  • usePasswordVisibility(): {
        showPassword: boolean;
        togglePasswordVisibility: (() => void);
    }

  • Custom hook for managing password visibility state

    This hook provides a reusable way to handle password visibility toggles across different forms, eliminating code duplication and providing consistent password field behavior throughout the application.

    The hook manages a boolean state for password visibility and provides a toggle function that can be easily integrated with input components. It's designed to work seamlessly with React Native Paper's TextInput component and other input libraries.

    Returns {
        showPassword: boolean;
        togglePasswordVisibility: (() => void);
    }

    Object containing visibility state and toggle function

    • showPassword: boolean
    • togglePasswordVisibility: (() => void)
        • (): void

        • Returns void

    Example

    // Basic usage with React Native Paper TextInput
    const { showPassword, togglePasswordVisibility } = usePasswordVisibility();

    <TextInput
      label="Password"
      secureTextEntry={!showPassword}
      right={
        <TextInput.Icon
          icon={showPassword ? "eye-off" : "eye"}
          onPress={togglePasswordVisibility}
        />
      }
    />
